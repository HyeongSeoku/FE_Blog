## 소개

Next.js는 **React 기반 프레임워크**로, SSR(Server-Side Rendering), CSR(Client-Side Rendering), 그리고 정적 사이트 생성(SSG) 기능을 통합적으로 제공하는 강력한 도구입니다. Next.js 14는 개발자 경험(DX)을 개선하고 성능을 극대화하기 위한 새로운 기능을 도입했습니다.

## Next.js 14의 주요 기능

### App Router와 Page Router 통합

Next.js 14에서는 기존 **Page Router**와 새롭게 도입된 **App Router**를 병렬로 사용할 수 있습니다. 이는 점진적인 마이그레이션을 가능하게 하며, 기존 프로젝트에서 App Router의 기능을 활용할 수 있도록 돕습니다.

#### 특징

- `app` 디렉토리 기반의 **파일 시스템 라우팅**.
- 서버 컴포넌트와 클라이언트 컴포넌트의 혼합 지원.
- Layout 및 중첩 라우팅 구조 지원.

#### 예제: App Router 기본 사용

```javascript
// src/app/layout.tsx
export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <body>{children}</body>
    </html>
  );
}
```
